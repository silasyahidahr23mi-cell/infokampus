PROGRAM STUDI MANAJEMEN INFORMATIKA

KEUNGGULAN PROGRAM STUDI MANAJEMEN INFORMATIIKA

A. Informatika Komputer

Tujuan Pendidikan

Menjadi tenaga ahli madya di bidang Software Enginering yg menghsilkan produk-produk inovatif dalam menerapkan proses bisnis berbasis tekhnologi informatika.
Standar Pendidikan:

  1. Menguasai bahasa inggris secara lisan maupun tulisan
  2. Menguasai komputer Microsoft office
  3. Terampil membuat aplikasi berbasis Claint Server
  4. Terampil membuat WEB Development Profesional

B. Multimedia Design

Tujuan Pendidikan

Menjadi tenaga ahli madya yang kompeten di bidang multimedia dari merancang design grafis, produksi multimedia dan videografi untuk kebutuhan komunikasi visual bisnis industri.
Standar Pendidikan:

  1. Menguasai bahasa inggris secara lisan maupun tulisan.
  2. Menguasai komputer Microsoft office.
  3. Terampil membuat design animasi, video editing dan multimedia interaktif.
  4. Terampil membuat WEB Development Profesional.
